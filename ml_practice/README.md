# 🏠 House Price Prediction using Simple Linear Regression

## 📌 Project Overview

This is a beginner Machine Learning project that predicts the price of a house based on its living area (square feet) using the Simple Linear Regression algorithm.

The project demonstrates the complete Machine Learning workflow, including data loading, model training, visualization, model saving with Pickle, and deployment using Streamlit.

---

## 🎯 Project Objective

The objective of this project is to:

- Understand Simple Linear Regression
- Train a Machine Learning model
- Predict house prices
- Save the trained model using Pickle
- Build a simple web application using Streamlit

## ▶️ How to Run the Project

### 1. Clone the repository

```bash
git clone https://github.com/your-username/house-price-prediction.git
```

### 2. Open the project folder

```bash
cd house-price-prediction
```

### 3. Install the required libraries

```bash
pip install -r requirements.txt
```

### 4. Train the model

```bash
python practice_house.py
```

### 5. Run the Streamlit application

```bash
streamlit run app.py
```
