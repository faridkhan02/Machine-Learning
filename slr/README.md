# 💼 Salary Prediction using Simple Linear Regression

## 📌 Project Overview

This beginner Machine Learning project predicts employee salaries based on years of experience using the Simple Linear Regression algorithm.

The project demonstrates the complete machine learning workflow, including data loading, model training, visualization, saving the trained model using Pickle, and deploying it with Streamlit.

---

## 🎯 Objectives

- Learn Simple Linear Regression
- Train a Machine Learning model
- Predict employee salaries
- Save the trained model using Pickle
- Build a Streamlit web application

## 🚀 How to Run

Install the required packages:

```bash
pip install -r requirements.txt
```

Run the training script:

```bash
python simple_linear_regression.py
```

Run the Streamlit application:

```bash
streamlit run app.py
```
